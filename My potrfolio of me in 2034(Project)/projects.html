My projects

Minimalistic Portfolio Website Design

This project showcases a modern, minimalistic portfolio website, designed for creatives who prefer a clean and elegant online presence. The layout features a centered hero section with a concise introduction and a call-to-action button, guiding visitors through the site with ease. The sleek, horizontal navbar ensures simple navigation, while the portfolio grid elegantly displays work with clean, bordered thumbnails. The neutral color scheme—white background, black text, and light gray accents—emphasizes simplicity and professionalism, making the content the focal point of the site.

AI-Powered Task Manager

This project is an AI-powered task management application designed to enhance productivity by intelligently organizing tasks based on priority, deadlines, and user behavior. Built with React on the front end and Node.js on the back end, it integrates AI algorithms to learn from user interactions, offering personalized suggestions and automating routine tasks. The app also features seamless cloud synchronization, ensuring that users can access their tasks from any device. This project showcases my ability to blend AI with user-centric design, creating a tool that adapts to individual workflows.

Car Dealership Website

This project is a comprehensive car dealership website designed to provide a seamless online car-buying experience. Built using React for the front end and Express.js with MongoDB for the back end, the website features an intuitive user interface that allows customers to browse, compare, and purchase vehicles with ease. The site includes advanced search and filtering options, detailed vehicle descriptions, high-resolution image galleries, and a secure online booking system. Additionally, it integrates a CRM system to manage customer inquiries and sales leads efficiently. This project demonstrates my ability to create user-friendly, data-driven web applications that cater to both business needs and customer satisfaction.
